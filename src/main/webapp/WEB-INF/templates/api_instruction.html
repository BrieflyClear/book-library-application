API

Endpoints:

To reach the endpoints go to the endpoint /api followed by the endpoints listed below:

  • /books
    • /isbn/{isbn}
      returns the book by given ISBN
    • /category/{category}
      returns a list of books by given category
  • /ratings
    returns a list of authors with their average ratings
  • /google
    uses Google Books API. The return list is always limited to 10 records
    • /search/{value}
      returns books by the search value
    • /search/{value}/limit/{maxResults}
      limits the above to the given amount (max 40)
    • /search/{value}/startIndex/{index}
      returns books by the search value. The startIndex determines the first index of the list
    • /search/{value}/startIndex/{index}/limit/{maxResults}
      limits the returned list to the given amount (max 40) with the first index at startIndex
